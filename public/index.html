            Settings
            

          
            
            
            AI assistant (inspector)
            
            
            
              Snappi does not store secrets in Settings. Configure via environment
              variables before launch:
            
            
            
              
              
            • 
                OpenAI cloud: OPENAI_API_KEY (or
                SNAPPI_OPENAI_API_KEY). Optional SNAPPI_AI_MODEL (default
                gpt-4o-mini).
              
              • 
              
            • 
                Local (Ollama): e.g.
                SNAPPI_OPENAI_BASE_URL=http://127.0.0.1:11434/v1 and
                SNAPPI_AI_MODEL=llama3.2 — no API key required. Or set
                SNAPPI_AI_LOCAL=1 with a localhost base URL.
              
              • 
            
            
          
            

          
            
            
            GitHub API token
            
            
            
              This app does not store a token in Settings. You
              do not type a token here.
            
            
            
            
              For the web workflow (npm run web,
              thread preview in the browser), set the environment variable
              GITHUB_TOKEN before starting the server if the API is
              required (rate limits or private data).
            
            
          
            

          
            
            
            Git & private repos
            
            
            
              Run PR dev server uses git clone on
              your machine. For private repositories, configure SSH keys or
              another credential helper so Git can access GitHub. That is
              separate from GITHUB_TOKEN.
